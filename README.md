# Email Wallet Frontend

Website for content creators

### Requirements
* nodejs v16.13.2 (recommended)
* mpn v8.1.2

### Commands 
* Fetch repo
```
git clone https://github.com/fegloff/exercise.git
```
* Install dependencies
```
yarn
```
* Development
```
yarn start
```
* Build
```
yarn build
```
The build artificats will be in /build folder.
